# @mechatronic/napi-window

Native NAPI binary for the mechatron window subsystem.

This package contains only prebuilt `.node` binaries and is intended to be
installed as an optional dependency of
[mechatron](https://www.npmjs.com/package/mechatron).  It has no JavaScript or
TypeScript of its own — all API surface is provided by the `mechatron` package.

See the [mechatron README](https://github.com/p120ph37/mechatron#readme) for
usage, platform support, and documentation.
